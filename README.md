## 使用方法 Use Guide

1. go [here](https://github.com/Over-Run/mc2d-acv) to download。  
首先，到 [此处](https://github.com/Over-Run/mc2d-acv) 下载。
2. Write a startup script and put in your directory.  
编写一个启动脚本并放到文件夹里。

### The startup script

```batch
@echo off
java -jar *.jar
pause
```
